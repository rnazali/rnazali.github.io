# Big Little Shelter 🏕️

Step by step on creating this site has been documented on [this article](https://rnazali.github.io/2023/08/02/creating-hassle-free-and-markdown-based-blog-with-jekyll.html).

## Development

To start developing, run:

```
bundle exec jekyll serve [--draft]
```

> Add `--draft` to make draft post appear

## Deployment

Push this repo to `jekyll` branch to trigger the build pipeline.
